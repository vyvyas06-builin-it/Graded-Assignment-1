# StaySmart Hotel Cancellation Prediction - Graded Assignment 1

This repository contains my submission for Graded Assignment 1. The objective of this project is to build a predictive model for hotel cancellations using the Hotel Bookings dataset, with a strong focus on feature engineering and preprocessing techniques rather than just model complexity.

## Project Structure

The repository is organized as follows:

* `FeatureEngineering_Capstone.ipynb`: The main Jupyter Notebook containing the code, analysis, and output for all 8 assigned tasks.
* `src/`: A directory intended for helper functions. To make the grader's job easier, most of the pipeline logic was kept directly inside the notebook so it can be read sequentially.
* `report/`: Contains `Report.pdf`, which includes the required graphs, my answers to the theoretical questions, and the final executive summary.
* `requirements.txt`: The list of Python dependencies required to execute the code.
